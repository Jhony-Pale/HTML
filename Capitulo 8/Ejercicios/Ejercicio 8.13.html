8.13: ¿Qué es lo que hace el siguiente programa?

switch ( k )
{
case 1:
break;
case 2:
case 3:
++k;
break;
case 4:
--k;
break;
default:
k *= 3;
} // fin de switch
x = k;

¿Qué valores se asignan a x cuando k tiene valores 1, 2, 3, 4 y 10?

Cuando k = 1:
x = 1

Cuando k = 2:
x = 3

Cuando k = 3:
x = 4

Cuando k = 4:
x = 3

Cuando k = 10:
x = 30
