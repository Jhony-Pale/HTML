7.4: Escriba instrucciones de JavaScript para realizar cada una de las siguientes tareas:

  1. Declarar las variables suma y x.
    Instrucción: var suma, x;

  2. Asignar 1 a la variable x.
    Instrucción: x = 1;

  3. Asignar 0 a la variable suma.
    Instrucción: suma = 0;

  4. Sumar la variable x a suma y asignar el resultado a la variable suma.
    Instrucción: suma = suma + x;

  5. Imprimir "La suma es: ", seguido del valor de la variable suma.
    Instrucción: document.writeln( "La suma es: ", + suma );
