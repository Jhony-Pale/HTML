7.6: Determine el valor de cada variable después de realizar el cálculo. Suponga que, cuando se empieza a ejecutar cada instrucción, todas las variables tienen el valor entero 5.

  1. producto *= x++;
    Respuesta: producto = 25, x = 6.

  2. cociente /= ++x;
    Respuesta: cociente = 0.83333, x = 6.
