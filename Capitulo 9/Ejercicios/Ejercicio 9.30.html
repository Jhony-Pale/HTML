9.30: ¿Qué hace la siguiente función?

  1. // El parámetro b debe ser entero
    // positivo para evitar recursividad infinita
    function misterio( a, b )
    {
     if ( b == 1 )
      return a;
     else
      return a + misterio(a, b - 1);
    }


    Respuesta: Regresa a * b, o sea, la suma de a un total de b cantidad de veces.
