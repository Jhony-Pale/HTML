10.8: Escriba instrucciones de JavaScript para realizar cada una de las siguientes tareas:

  1. Mostrar el valor del séptimo elemento del arreglo f.
    Instrucción:
    document.writeln(f[6]);

  2. Inicializar cada uno de los cinco elementos del arreglo unidimensional g en 8.
    Instrucción:
    var g = [8,8,8,8,8];

  3. Calcular el total de los elementos del arreglo c, que contiene 100 elementos numéricos.
    Instrucción:
    var total = 0;
    for ( var a in c ){
     total += c[a];
    }

  4. Copiar el arreglo a de 11 elementos en la primera porción del arreglo b, que contiene 34 elementos.
    Instrucción:
    for( var x in a ){
     b[x] = a[x];
    }

  5. Determinar e imprimir el valor más pequeño y el más grande contenido en el arreglo w de 99 elementos de punto flotante.
    Instrucción:
    document.writeln(Math.min(...w));
    document.writeln(Math.max(...w));
