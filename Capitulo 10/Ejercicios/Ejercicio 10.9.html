10.9: Considere un arreglo de dos por tres que almacena enteros.

  1. Escriba una instrucción para declarar y crear el arreglo t..
    Instrucción:
    var t = new Array( new Array(3), new Array(3) );

  2. ¿Cuántas filas tiene t?.
    2 filas.

  3. ¿Cuántas columnas tiene t?.
    3 columnas.

  4. ¿Cuántos elementos tiene t?.
    6 elementos.

  5. Escriba los nombres de todos los elementos en la fila 1 de t.
    t[0][0], t[0][1], t[0][2].

  6. Escriba los nombres de todos los elementos en la tercera columna de t.
    t[0][2], t[1][2].

  7. Escriba una sola instrucción para establecer el elemento de t en la fila 1 y la columna 2 en cero.
    Instrucción:
    t[0][1] = 0;

  8. Escriba una serie de instrucciones para inicializar cada elemento de t en cero.
    Instrucción:
    t[0][0] = 0;
    t[0][1] = 0;
    t[0][2] = 0;
    t[1][0] = 0;
    t[1][1] = 0;
    t[1][2] = 0;

  9. Escriba una instrucción for anidada para inicializar cada elemento de t en cero.
    Instrucción:
    for(var x in t){
     for(var y in t[x]){
      t[x][y] = 0;
     }
    }

  10. Escriba una serie de instrucciones para determinar e imprimir el valor más pequeño en el arreglo t.
    Instrucción:
    document.writeln(Math.min(...t));

  11. Escriba una instrucción sin repetición para mostrar los elementos de la primera fila de t.
    Instrucción:
    document.writeln(t[0][0]);
    document.writeln(t[0][1]);
    document.writeln(t[0][2]);

  12. Escriba una serie de instrucciones para imprimir el arreglo t en un formato tabular ordenado. Liste los índices de columnas como encabezados a lo largo de la parte superior y liste los índices de fila en la parte izquierda de cada fila.
    Instrucción:
    var t = [
     [1,2,3],
     [4,5,6]
    ];
    document.writeln("<table><thead><th></th><th>0</th><th>1</th><th>2</th><tbody>");
    for(var x in t){
     document.writeln("<tr><td>" + x + "</td>");
     for(var y in t[x]){
      document.writeln("<td>" + t[x][y] + "</td>");
     }
    }
    document.writeln("</tbody></table>");
