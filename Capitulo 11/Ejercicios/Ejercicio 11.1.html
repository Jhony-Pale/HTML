11.1: Complete las siguientes oraciones.

  1. Como JavaScript usa objetos para realizar muchas tareas, se conoce por lo común como lenguaje de programación basado en objetos.

  2. Todos los objetos tienen atributos y exhiben comportamientos.

  3. Los métodos del objeto Mathnos permiten realizar muchos cálculos matemáticos comunes.

  4. Al proceso de invocar (o llamar) al método de un objeto se le conoce como enviar un mensaje al objeto.

  5. Las literales de cadena o constantes de cadena se escriben como una secuencia de caracteres en comillas dobles o comillas sencillas.

  6. Los índices para los caracteres en una cadena empiezan en cero (0).

  7. Los métodos indexOf y lastindexOf de String buscan la primera y última ocurrencias de una subcadena en un objeto String, respectivamente.

  8. Al proceso de dividir una cadena en tokens se le denomina tokenización.

  9. El procesamiento de fechas y horas puede realizarse con base en la zona horaria local de la computadora o en el Tiempo Universal Coordinado (UTC) del Estándar de Tiempo Mundial.

  10. El método parse de Date recibe como argumento una cadena que representa una fecha y hora, y devuelve el número de milisegundos transcurridos entre la medianoche del 1 de enero de 1970 y la fecha y hora especificadas.

  11. Las aplicaciones Web pueden usar la propiedad localStorage del objeto window para almacenar hasta varios megabytes de datos de cadenas de pares clave/valor en la computadora del usuario y pueden acceder a estos datos a través de sesiones y pestañas del navegador.

  12. Las aplicaciones Web que necesitan acceso a los datos de pares clave/valor para sólo una sesión de navegación y que deben mantener esos datos separados entre varias pestañas pueden usar la propiedad sessionStorage del objeto window.

  13. Un selector first-child de CSS3 selecciona el primer hijo de un elemento.

  14. Un selector nth-child de CSS3 con el argumento "odd" selecciones los elementos hijos impares y uno con el argumento "even" selecciona los elementos hijos pares.
